Defensibility

"Won't Anthropic Build This?"

1. Different problems
Auto mode is an input gate — it checks what the agent wants to do. The top user problems are output problems: lying, context rot, test cheating, false completion claims. Luthien monitors what the agent actually produces.
2. Fragmenting market
OpenCode (15K+ stars), Aider (30K+), Codex, OpenClaw — the coding agent market is fragmenting fast. Anthropic won't build management tooling for competitors' clients. Luthien is provider-agnostic.
3. Anthropic agrees
Anthropic researcher, April 2026: "The long-term solution probably looks more like something proxy-side" and "the space for control protocols is quite big."

"We want labs to copy us. That's our theory of change."
