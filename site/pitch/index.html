Defensibility

"Won't Anthropic Build This?"

1. Different problems
Auto mode is an input gate. It checks what the agent wants to do. But the top user complaints are output problems: lying, context rot, test cheating, false completion claims. Luthien monitors what the agent actually produces.
2. Fragmenting market
OpenCode (15K+ stars), Aider (30K+), Codex, OpenClaw. The coding agent market is fragmenting fast. Anthropic won't build management tooling for competitors' clients. Luthien is provider-agnostic.
