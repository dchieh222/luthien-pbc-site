Problema: Desarrolladores

El potencial desperdiciado de Claude Code

Los desarrolladores están frustrados: no pueden dejar que Claude Code trabaje solo, así que lo vigilan constantemente.

Claude Code puede funcionar autónomamente durante 12 horas. En la práctica, los humanos lo supervisan cada 45 segundos porque los riesgos de dejarlo solo son demasiado altos: violaciones de cumplimiento, errores técnicos, vulnerabilidades de seguridad, o simplemente no seguir instrucciones.

Autonomía de 12 horas: Documentación de Claude Code, Anthropic. Supervisión cada 45 segundos: observado en más de 20 llamadas de descubrimiento con desarrolladores que usan Claude Code 30+ hrs/semana (investigación interna de Luthien, Mar-Abr 2026).

Defensibilidad

¿No construirá Anthropic esto?

1. Ellos reconocieron que nuestro enfoque es la solución a largo plazo
"The long-term solution probably looks more like something proxy-side" y "the space for control protocols is quite big." (Fabien Roger, Anthropic, 1 abr 2026)
2. No tiene sentido que los laboratorios construyan herramientas agnósticas de proveedor
El mercado de agentes de código se está fragmentando rápidamente: OpenCode, Codex, Cursor, OpenClaw. Anthropic no va a construir herramientas de gestión para los clientes de sus competidores.
3. Ayudamos a impulsar la adopción empresarial de Claude Code
Los gateways de LLM ya son un estándar de infraestructura empresarial, y Claude Code está diseñado para funcionar con ellos. Anthropic se beneficia de lo que estamos construyendo: acelera la adopción empresarial de Claude Code.
4. Si lo hacen, excelente: defensa en profundidad
Más capas de seguridad es el objetivo. Y nosotros nos enfocaremos en Codex, Cursor, OpenCode, OpenClaw, y todo lo demás para lo que ellos no construirán.

Distribución

La misma estrategia que construyó Auth0.

Estamos construyendo control y cumplimiento de IA que a los desarrolladores les encanta.

$6.5B

Auth0 fue fundada en 2013 como un SDK de identidad para desarrolladores. En mayo de 2021, Okta la adquirió por $6.5B, a pesar de que Okta ya vendía su propio producto de identidad para clientes. Auth0 había construido "a developer first bottoms up acquisition strategy."1

1 Advait Ruia, "The Real Reason Okta Spent $6.5B on Auth0," SuperTokens, 5 de marzo de 2021. La valoración previa de Auth0 era $1.9B en su Serie F de julio 2020, por lo que la adquisición fue aproximadamente 3.5x ese precio. El acuerdo también duplicó el mercado total accesible de Okta de aproximadamente $30B (identidad laboral) a $55B (identidad laboral más identidad de clientes).
